# SEO & Website Performance Reporting Dashboard (GA4 + Looker Studio)

Built an interactive reporting dashboard using Google Analytics 4 demo ecommerce data to analyze website traffic, SEO performance, user behavior, and conversion effectiveness. The goal was to simulate a real-world Reporting Analyst workflow: auditing acquisition channels, identifying drop-offs, and recommending optimization actions.

## Tools Used

Google Analytics 4 (GA4)

Looker Studio

## Key Metrics Analyzed

Sessions: 58,964

Users: 40,454

Purchase Conversion Rate: ~1%

Traffic Sources: Direct, Organic, Paid, Email

Landing Page Bounce Rate

Keyword impressions, CTR, average position

## Insights & Recommendations

66% of traffic driven by Direct + Organic → strong SEO/brand presence

Checkout page bounce rate (51%+) indicates purchase friction

Conversion rate (~1%) below ecommerce benchmarks → UX optimization needed

High-impression, low-CTR keywords present quick SEO wins

Email traffic shows higher engagement → scale lifecycle campaigns

## Analysis Performed

Traffic source contribution and channel mix analysis

Landing page performance & bounce rate diagnostics

Checkout funnel drop-off detection

SEO opportunity identification using impressions vs CTR

Engagement scoring & behavior-based segmentation

## Outcome

This dashboard is live and publicly accessible — built to simulate a production-grade reporting workflow for a digital marketing team.
